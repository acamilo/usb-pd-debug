# Component Selection

## Processor
Datasheet shown in stream.
* STM32F078VB

## Hub
The hub in the stream is EOL, Picked a different one
* CY7C65621-56LTXC

## Level Shifters
Picked the shifters from glasgow
* SN74LVC1T45DCKR

## USB Mux
* FSUSB73UMX

## PD Chip
* FUSB302

## LDO
Can handle 20v. Can output 1.8 and 3.3
* NCP718BSNADJT1G

## Power Switching
* TCK323G
